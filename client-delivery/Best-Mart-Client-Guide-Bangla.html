১. Short overview

Best Mart হলো Dubai-focused bilingual ecommerce website। Customer English/Arabic storefront ব্যবহার করে product browse, color/size variant select, cart/checkout, COD/Stripe payment, order tracking এবং account order history দেখতে পারে।

Customer side

Home, shop, product details, cart, checkout, account, order history, order tracking, privacy/terms.

Admin side

Dashboard, products, add product, categories, banners, orders, users, reviews, coupons, settings, payment/courier controls.

Dubai controls

Shipping area dropdown, delivery slot, apartment/tower field, AED pricing, VAT/TRN invoice support, COD controls.

Notifications

Header notification, new product notification, order email, admin notification counters, optional WhatsApp/SMS flow.

৪. Admin workflow guidance

নতুন product add: Admin Dashboard → Add product → category select → product information → color image/variant → size/stock → price/SEO → Save.
Color-wise product: প্রতিটি color-এর আলাদা image, stock এবং optional size stock দিন। Customer product details-এ color tap করলে সেই color image দেখবে।
Order process: Orders page → order open → status Confirmed/Processing/Shipped/Delivered করুন। User account order history এবং order confirmation page-এ updated status দেখবে।
Payment control: Settings → Payment controls → Cash on delivery বা Stripe visible/hidden করুন। Stripe key admin থেকে set করা যাবে, কিন্তু secret key public করা যাবে না।
Shipping area: Settings → Currency and shipping থেকে Dubai/UAE emirate fee, free shipping threshold, delivery days এবং COD availability update করুন। Checkout page-এ dropdown দেখাবে।
Courier setup: Settings → Dubai courier integration → provider, account number, API key/secret, webhook secret, tracking URL template save করুন। Provider final হলে developer এই saved config দিয়ে live booking/tracking API connect করবে।
Homepage control: Banners, Categories, Featured products এবং Settings থেকে homepage content, category order, trust message এবং announcement control করা যাবে।
Notification check: Dashboard notification number দেখে pending orders, reviews, low stock এবং customer-facing new product notification verify করতে হবে।

৬. Payment, courier, and secret note

Important: Stripe secret key, SMTP password, Cloudinary API secret, courier API secret, NextAuth secret এবং database URL কখনো client-facing screenshot, GitHub, frontend code বা public PDF-এ দিবেন না। Admin panel password fields-এ value save করা যাবে, কিন্তু client handover document-এ actual secret লিখবেন না।
  • COD: Dubai delivery business-এর জন্য default cash-on-delivery option রাখা হয়েছে।
  • Stripe: Admin settings থেকে visible/hidden, display name, publishable key, secret key, webhook secret এবং checkout mode set করা যায়।
  • Courier: Manual, Aramex UAE, Fetchr, DHL, Shiprocket/partner অথবা other courier option save করা যায়। Live courier account approve হলে API integration করা যাবে।
  • Email: SMTP set করলে order placed/status update email customer-এর কাছে যাবে।

৭. Client acceptance checklist

  • Homepage brand, category, featured products এবং trust section ঠিক আছে।
  • Shop color filter, product card spacing, mobile 2-column grid ঠিক আছে।
  • Product details color image switch এবং size/stock variant ঠিক আছে।
  • Checkout guest order, apartment/tower, shipping area dropdown, delivery slot ঠিক আছে।
  • Admin order status update করলে user order history/status ঠিকভাবে update হয়।
  • Admin dashboard notification number এবং sidebar routing বোঝা সহজ।
  • Settings থেকে COD/Stripe/courier/UI/SEO/shipping controls manage করা যায়।
  • PDF guide অন্য device-এ image missing ছাড়া open হয়।
