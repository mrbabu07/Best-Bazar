১. আগে যেগুলো confirm করতে হবে

Hostever plan check: cPanel dashboard-এ Setup Node.js App, Node.js Selector, অথবা Application Manager option থাকতে হবে। যদি না থাকে, shared cPanel plan-এ Next.js server run হবে না; তখন Hostever VPS/Node-supported plan নিতে হবে।

Server requirement

Node.js 18+ বা 20+, npm, terminal access, environment variables, custom startup file support.

Database requirement

এই project PostgreSQL ব্যবহার করে। সবচেয়ে safe option হলো Neon PostgreSQL external database। cPanel MySQL দিয়ে এই project চলবে না।

Domain requirement

Domain DNS Hostever nameserver অথবা cPanel Zone Editor দিয়ে hosting server-এ point থাকতে হবে। SSL AutoSSL/Let's Encrypt enable করতে হবে।

Secret requirement

.env.local values কখনো GitHub বা public file-এ upload করবেন না। cPanel Environment Variables panel-এ বসাবেন।

৩. Local project ready করবেন

Latest code pull: Local PC-তে project folder open করে latest GitHub code pull করুন।
Dependency install: Terminal-এ run করুন: 
npm install
Environment test: Local .env.local ঠিক আছে কিনা check করুন। বিশেষ করে DATABASE_URL, NEXTAUTH_SECRET, NEXTAUTH_URL, NEXT_PUBLIC_SITE_URL।
Build test: Local build pass করান: 
npm run type-check
npm run build
Admin user ensure: Production database connect করা থাকলে run করুন: 
npm run admin:ensure

৪. Database setup - Neon PostgreSQL

Recommended: Hostever cPanel-এর local MySQL/PostgreSQL-এর বদলে Neon PostgreSQL ব্যবহার করুন। Project already Prisma PostgreSQL schema দিয়ে তৈরি।
Neon dashboard-এ login করুন এবং project/database create করুন। Region হিসেবে যতটা সম্ভব app users-এর কাছে বা hosting-এর কাছে region select করুন।
Connection string copy করুন। Pooler URL ব্যবহার করুন। Example format: 
postgresql://USER:PASSWORD@HOST-pooler.REGION.aws.neon.tech/DB?sslmode=require&channel_binding=require
Connection URL-এর শেষে performance parameters যোগ করুন: 
&connection_limit=5&pool_timeout=20&connect_timeout=10
Local বা cPanel terminal থেকে schema push করুন: 
npm run db:generate
npm run db:push
Initial data/admin দরকার হলে seed/admin command run করুন: 
npm run db:seed
npm run admin:ensure

৫. cPanel file upload method

Best method: Git

cPanel-এ Git Version Control থাকলে GitHub repository clone করুন। Deploy/update সবচেয়ে clean হবে।

Alternative: ZIP upload

Local project zip করে File Manager দিয়ে upload করুন। node_modules, .git, .env.local upload না করাই ভালো।

Upload করার সময় এই files/folders দরকার:

ItemUpload?Reason
app, components, lib, store, utilsYesNext.js source code
publicYesImages, icons, manifest, PWA files
prismaYesDatabase schema and seed
package.json, package-lock.jsonYesDependency install scripts
server.jsYesProduction startup file for Node app
next.config.mjs, tsconfig.json, tailwind.config.*YesBuild/config files
.env.localNoSecrets must be set from cPanel environment variables
node_modules, .nextNo preferredServer should install/build fresh. Upload only if cPanel has no build terminal.

৬. cPanel Node.js App create করবেন

cPanel login করুন → Setup Node.js App open করুন।
Create Application চাপুন।
Node version select করুন: Node.js 20 preferred, না থাকলে Node.js 18।
Application mode: Production।
Application root: যেখানে project upload করেছেন, যেমন best-mart অথবা bestmart-app।
Application URL: main domain বা subdomain select করুন, যেমন https://bestmart.ae।
Application startup file: server.js। এই project-এর npm start command node server.js চালায়।
Save/Create দিন। তারপর Run NPM Install অথবা terminal থেকে install/build run করুন।

৮. cPanel terminal commands

cPanel Terminal বা SSH access থাকলে application root folder-এ গিয়ে নিচের command run করুন:

npm install
npm run db:generate
npm run db:push
npm run build
npm run admin:ensure

তারপর cPanel Node.js app panel থেকে Restart দিন।

যদি terminal না থাকে, Hostever support-কে বলতে হবে Node.js app root-এ npm install, npm run build, npm run db:push, npm run admin:ensure run করে দিতে।

৯. Domain and SSL setup

Domain Hostever nameserver-এ point করুন অথবা DNS A record cPanel server IP-তে দিন।
cPanel → Domains/Subdomains থেকে domain/subdomain add করুন।
Node.js Application URL হিসেবে same domain select করুন।
cPanel → SSL/TLS Status অথবা AutoSSL থেকে SSL issue করুন।
NEXTAUTH_URL এবং NEXT_PUBLIC_SITE_URL অবশ্যই HTTPS domain দিয়ে update করুন।
Node app restart করুন। Browser cache clear করে site test করুন।

১১. Update/deploy new version

GitHub-এ latest code push করুন।
cPanel Git Version Control থাকলে Pull/Deploy করুন। ZIP method হলে old files backup নিয়ে new ZIP upload/extract করুন।
Dependency change থাকলে npm install run করুন।
Database schema change থাকলে npm run db:push run করুন।
Always run npm run build and restart Node app।
Home, checkout, admin dashboard smoke test করুন।

১৩. Final handover checklist

  • Hostever cPanel Node.js support confirmed.
  • Domain and SSL active.
  • All env variables set in cPanel, not in public files.
  • Neon database connected and schema pushed.
  • Admin user created and login tested.
  • Home, shop, product, cart, checkout, order confirmation tested.
  • Admin product, order, shipping/payment settings tested.
  • Email, Stripe, Cloudinary configured if client needs them live now.
  • Backup plan documented: GitHub repo + Neon backup/export + cPanel file backup.
